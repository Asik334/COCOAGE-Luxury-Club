Премиальный сайт косметологической клиники **COCOAGE Luxury Club** (г. Актобе).

## 🌐 Live сайт

Задеплоено на Vercel → [cocoage.vercel.app](https://cocoage.vercel.app)

## 🗂 Структура

```
cocoage/
├── index.html        # Главная страница
├── css/
│   └── style.css     # Стили (CSS custom properties, анимации)
├── js/
│   └── main.js       # JS: скролл, бургер, IntersectionObserver
├── vercel.json       # Конфиг деплоя
└── README.md
```

## 🚀 Деплой на Vercel

1. Залить репо на GitHub
2. Войти на [vercel.com](https://vercel.com) → **New Project**
3. Импортировать репо → Vercel автоматически определит статический сайт
4. Нажать **Deploy**
